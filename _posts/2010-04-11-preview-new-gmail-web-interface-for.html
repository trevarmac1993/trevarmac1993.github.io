--- layout: post title: "Preview the new Gmail web interface for the iPad in Chrome" date: 2010-04-11 comments: true tags: [ annotated, Google ] --- Since the launch of the iPhone and later the Android mobile OS, Google has investigated some time in creating more mobile friendly interfaces of their common web applications (Gmail, GReader, GDocs,Wave), optimized for the advanced touch interfaces. Given the buzz about the launch of the iPad recently, it is no surprise that they have optimized their web interface to nicely fit the larger iPad touch screen.

You can actually look at the live interface by using your Chrome browser.

If you experience any problems try to restart Chrome or delete all of the cookies. Google does couple of redirects to the different UIs based on user agents, but cookies may cause to start an endless redirect loop.

Here are some references on how you can launch the iPad Gmail interface (Firefox doesn't work too good since it's not a Webkit engine):

Chrome: http://googlesystem.blogspot.com/2010/04/use-gmail-for-ipad-in-google-chrome.html
