--- layout: post title: "Gmail awakes attention" date: 2008-06-06 comments: true tags: [ Google ] --- After quite a long time of silence, today the were a lot of rumors regarding Gmail. After adding an initial loading page, they released a "labs" tab today, which is available in Gmail's settings page. Users can activate and try out new features for their Gmail account there and give immediate feedback. I find it an interesting way of developing new features in an agile fashion by getting continuous feedback from users which is the key success factor. It's nice because with those "labs" features, the Gmail team can at any time just release a new feature, let the users try them out and see how they react to them. If it is a success, development will go on and otherwise it will just be taken away. Things can be released in a very informal fashion without any commitment.
